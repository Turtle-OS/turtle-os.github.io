# Olá a todos!

TurtleOS é um projeto de simulador de OS em Python, feito para testes, educação e muitos outros!
Vamos fazer uma introdução simples do Python
Python é uma linguagem de programação, muito usada para LLMs, projetos, cibersegurança
Um exemplo, definindo uma variável e imprimindo ela

```python
VAR = "Olá, mundo!"  
  
print(VAR)
```

## Instalando o Python

### No Microsoft™ Windows:

#### Abra o PowerShell e digite:
```pwsh
winget install Python.Python.3
```
### No Android (Termux):
#### Abra o Termux e digite
```bash
pkg install python3
```
### No macOS:
> [!NOTE]
> O comando brew precisa estar instalado. Em versões mais novas do macOS, ele já está presente de forma integrada ao sistema.
> Caso você nao tenha o comando brew, por favor instale-o
